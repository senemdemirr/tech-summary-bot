ASP.NET Core 2.3 Desteği Sona Eriyor

Microsoft, ASP.NET Core 2.3 için desteğin 7 Nisan 2027 tarihinde sona ereceğini duyurdu. Bu tarihten sonra, ASP.NET Core 2.3 için güvenlik yamaları, hata düzeltmeleri veya teknik destek sağlanmayacak. Bu duyuru, Microsoft Destek Yaşam Döngüsü Politikası kapsamında, "Araçlar" olarak sınıflandırılan ürünler için gerekli olan 12 aylık ön bildirim süresini sağlamaktadır.

ASP.NET Core 2.3 paketleri, şu anda .NET Framework üzerinde, bu .NET Framework sürümlerinin destek döngüsüne uygun olarak desteklenmektedir. Ancak, 7 Nisan 2027'den sonra bu destek, kullanılan .NET Framework sürümünden bağımsız olarak sona erecektir. ASP.NET Core 2.3 paketleri, Entity Framework 2.3 paketlerini de içermekte olup, bu paketler de aynı tarihte destek dışı kalacaktır.

Büyük Resim: Bu İçerik Ne Anlatıyor ve Neden Önemli?

Bu duyuru, ASP.NET Core 2.3 kullanıcılarını, yaklaşan destek sonu hakkında bilgilendirmeyi amaçlamaktadır. Destek sona erdiğinde, bu sürüm için herhangi bir güvenlik güncellemesi veya teknik destek sağlanmayacak. Bu durum, kullanıcıların güvenlik açıklarına maruz kalma riskini artırabilir. Bu nedenle, kullanıcıların daha yeni ve desteklenen bir .NET sürümüne geçmeleri önerilmektedir.

Teknik Detaylar / Öne Çıkan Noktalar

ASP.NET Core 2.3'ün destek sonu, mevcut uygulamaların çalışmasını etkilemeyecek. Ancak, bu sürüm için yeni güvenlik güncellemeleri yayınlanmayacak. Ayrıca, teknik destek sağlanmayacak ve ASP.NET Core 2.3 paketleri kullanımdan kaldırılacak. Kullanıcıların, daha modern ve desteklenen bir .NET sürümüne geçmeleri önerilmektedir. Bu geçiş, performans, güvenlik ve geliştirici verimliliği açısından önemli iyileştirmeler sunmaktadır.

Etkiler: Kimleri Nasıl Etkiler?

ASP.NET Core 2.3 kullanıcıları, destek sonu nedeniyle güvenlik ve teknik destek konusunda risk altında olabilirler. Mevcut uygulamalar çalışmaya devam edecek olsa da, güvenlik açıklarına karşı savunmasız kalabilirler. Bu nedenle, kullanıcıların daha yeni bir .NET sürümüne geçmeleri önerilmektedir. Bu geçiş, uygulamaların güvenliğini ve performansını artırabilir.

Kısa Özet

  • ASP.NET Core 2.3 desteği 7 Nisan 2027'de sona erecek.
  • Desteğin sona ermesi, mevcut uygulamaların çalışmasını etkilemeyecek.
  • Güvenlik güncellemeleri ve teknik destek sağlanmayacak.
  • Kullanıcılar, daha modern bir .NET sürümüne geçmelidir.
  • Geçişte GitHub Copilot modernizasyonu gibi araçlar kullanılabilir.

Kapanış

ASP.NET Core 2.3 için desteğin sona ereceği tarih yaklaşıyor. Kullanıcıların, daha güvenli ve performanslı bir deneyim için, desteklenen bir .NET sürümüne geçiş yapmaları şiddetle tavsiye edilmektedir. Geçiş sürecini kolaylaştırmak için GitHub Copilot modernizasyonu gibi araçlar kullanılabilir.
